# 🦋 Lumiê Studio — Beleza & Estética

Site institucional com sistema de agendamento online para o Lumiê Studio.

## Stack

- **React 19** + **TypeScript** + **Vite 8**
- **Zustand** (estado global + persistência)
- **React Router v7**
- **date-fns** (manipulação de datas em pt-BR)
- Arquitetura **MVVM** (Models → ViewModels → Views)

## Rodar localmente

```bash
npm install
npm run dev
```
